"Show me all users created this month with paid orders"

SQL Query

SELECT
  u.id, u.email, u.created_at,
  COUNT(o.id) as order_count,
  SUM(o.amount) as total_spent
FROM users u
JOIN orders o ON u.id = o.user_id
WHERE u.created_at >= DATE_TRUNC('month', CURRENT_DATE)
  AND o.status = 'paid'
GROUP BY u.id, u.email, u.created_at
ORDER BY total_spent DESC;

psychology AI Explanation

This query finds users created this month who have at least one paid order. It joins the users and orders tables, filters for current month creation and 'paid' status, then aggregates the total spent per user, sorted by highest spenders first.
