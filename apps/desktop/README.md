# Aratu Desktop

This directory will contain the Electron desktop application.

- `src/main`: trusted Electron main process and sidecar lifecycle.
- `src/preload`: narrow, typed API exposed to the renderer.
- `src/renderer`: React and TypeScript user interface.

No runtime has been initialized yet. Phase 1 must select and record the Electron/Vite scaffolding, package manager, security defaults, and development workflow before application code is added.
